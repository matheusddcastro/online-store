# Dubray Books Online Store
This is a project was done for my course in Rathmines College of Further Education and 
showcases 3-tier Architecture: Presentation Tier, Server Tier, and Data Tier. 

Everything was done via Eclipse IDE, including the HTML and CSS code. The website design is quite simple
and static as there was no need for JavaScript for this specific project.

This project allows you to register a new user and log in to your account. It uses
MySQL database and Apache Tomcat Server.
